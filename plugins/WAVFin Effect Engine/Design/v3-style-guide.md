# Style Guide v2 (WAVFin Brand Identity)

## Brand Palette
- **Primary:** #334155 (Slate)
- **Primary Light:** #0047ab (Blue)
- **Primary Dark:** #00A59C (Teal)
- **Secondary:** #06b6d4 (Cyan)
- **Secondary Light:** #22d3ee (Light Cyan)
- **Accent:** #f59e0b (Amber/Orange)

## Theme Colors (Dark Mode)
- **Background Main:** #0f172a (Deep Navy)
- **Background Secondary:** #1e293b (Dark Slate) - Used for module cards
- **Background Tertiary:** #334155 (Slate) - Used for borders/tracks

## Typography
- **Primary Text:** #ffffff (White)
- **Secondary Text:** #e2e8f0 (Light Gray)
- **Muted Text:** #a1a1aa (Gray)
- **Font Stack:** System UI (San Francisco / Segoe UI / Roboto)

## Visual Components
- **Knobs:** Amber (#f59e0b) active arc on Slate (#334155) track.
- **Toggles:** Amber glow when active.
- **Glassmorphism:**
    - Background: `rgba(30, 41, 59, 0.6)`
    - Blur: `12px`
    - Border: `1px solid rgba(255, 255, 255, 0.08)`
